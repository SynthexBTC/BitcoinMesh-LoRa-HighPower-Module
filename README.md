# BitcoinMesh LoRa HighPower Module

Bitcoin Becomes Truly Uncensorable<br>
Broadcast Bitcoin transactions over LoRa mesh networks<br>
No Internet required at the sender's location<br>

- Connection via Bluetooth
- OLED Screen
- UltraLow-Power nRF52840 SoC
- E22-900M30S (SX1262) HighPower 1W LoRa Wireless Module
- On Board DC-DC regulator
- Ready for MESHTASTIC and MESHCORE firmwares
- Automatic humidity and temperature telemetry
- Can be mounted in a PVC pipe directly on the roof with the antenna
